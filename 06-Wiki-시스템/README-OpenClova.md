# Wiki 시스템 - OpenClova용 설명서

## 개요

이 Wiki 시스템은 **Hermes Agent**와 **Obsidian**을 활용한 개인 지식 관리 시스템입니다.
OpenClova와 Hermes Agent가함께하여 문서를 관리합니다.

---

## 디렉토리 구조

```
06-Wiki-시스템/
├── Raw/              ← 원본 자료 (사람이 수집)
├── Wiki/             ← AI가 정리한 문서
│   ├── 인덱스.md      → 전체 문서 목록 (자동 생성)
│   ├── 개념/          → 개념 정의
│   ├── 개체/          → 기업, 인물, 제품 등
│   ├── 주제/          → 프로젝트, 기술 스택 등
│   └── 요약/          → 긴 문서의 요약
├── Logs/             ← 활동 로그
├── README.md         ← 시스템 설명
└── 클로드.md         ← LLM 규칙 (매우 중요)
```

---

## 핵심 원칙

### 1. 수집은 사람이, 정리는 AI가
- **사람:** Raw 폴더에 원본 자료를放入
- **AI:** Raw → Wiki 변환 (분류, 정리, Wikilink 연결)

### 2. Wiki 문서는 반드시 출처 표시
```
출처: [[Raw/파일명]]
```

### 3. 60일 이상된 문서는 점검 대상
- `#오래됨` 태그 자동 추가
- Lint 스크립트가자동으로 검사

---

## 사용 명령어

### Ingest (수집/변환)
```
Raw 폴더의 파일을 읽고 Wiki 폴더에 정리된 문서로 변환
- 파일 분류 (개념, 개체, 주제, 요약)
- Wikilink 자동 연결
- 태그 자동 추가
```

### Query (검색/답변)
```
Wiki에서 정보를 검색하고 답변 생성
- 파일 이름, 내용, 태그에서 검색
- 반드시 출처 명시
```

### Lint (건강성 검사)
```
Wiki 문서의 건강 상태 점검
- 깨진 링크 검사
- 이중 타이틀 검사
- 빈 문서 검사
- 고아 문서 검사 (어떤 문서에서도 참조되지 않음)
- 오래된 문서 검사 (60일 이상)
```

---

## 문서 작성 규칙

### 필수 항목
1. **제목:** 파일名的과 일치하는 H1
2. **원본 정보:** 원본 파일명, 가져온 날짜
3. **요약:** 문서의 핵심 내용
4. **태그:** 관련 태그 (#개념 #프로젝트 등)
5. **출처:** [[Raw/파일명]] 형식으로 원본 명시

### Wikilink 사용
```markdown
다른 문서 연결: [[다른-문서-이름]]
출처 표시: [[Raw/원본-파일명]]
```

## Hermes Agent 설정

Hermes Agent에서 Wiki 시스템을 사용하려면:

1. **Vault 경로 인식**
   ```
   /home/comage/obsidian-vault/06-Wiki-시스템
   ```

2. **클로드.md 로드**
   모든 작업前에 클로드.md의 규칙을 확인

3. **명령어 실행**
   ```bash
   # 인제스트 (Raw → Wiki)
   bash ~/.hermes/scripts/wiki-ingest.sh

   # 검색
   bash ~/.hermes/scripts/wiki-query.sh "검색어"

   # 린트
   bash ~/.hermes/scripts/wiki-lint.sh
   ```
