# Hermes Wiki 지식 시스템

## 개요
Andrej Karparthi 방식의 개인 지식 위키 시스템

## 폴더 구조
```
06-Wiki-시스템/
├── Raw/           ← 원본 자료 (유튜브 자막, 문서, 세션 로그 등)
├── Wiki/          ← 정리된 지식 (개념, 개체, 요약)
│   ├── 개념/
│   ├── 개체/
│   └── 요약/
├── Logs/          ← 활동 로그
└── README.md      ← 이 파일
```

## 3단계 원칙
1. **Raw** → 원본 자료 저장
2. **Claude Code가 읽고** → 분류하고 연결하고
3. **Wiki** → 정리된 지식 데이터베이스 완성

## 사용법
- 세션结束时: Raw에 작업 내용 저장
- 매일 Cronjob: 분석 → Wiki에 정리
- 질문 시: Wiki 기반 답변

## 자동화
- Cronjob: 매일 아침 9시 인제스트 실행
- Cronjob: 매일 밤 10시 린트 실행
