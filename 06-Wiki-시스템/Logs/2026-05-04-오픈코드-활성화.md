# OpenCode 활성화 기록

날짜: 2026-05-04
상태: ✅ 성공

## 문제 원인

OpenCode CLI에서 `opencode run` 실행 시 `Missing Authentication header` 오류 발생

**원인:** `~/.config/opencode/opencode.json`의 `apiKey` 설정이 `{env:OPENROUTER_API_KEY}` 참조 형태인데, 환경변수 `OPENROUTER_API_KEY`가 세션에서 로드되지 않음

## 해결 방법

```bash
# ~/.bashrc에서 OPENROUTER_API_KEY를 source해서 실행
bash -c 'eval $(grep OPENROUTER ~/.bashrc | head -1) && opencode run "작업 내용" --dir /tmp'
```
