# KPP 파렛트 관리 (로지스올 WPPS)

## 로그인 및 접속

- 사이트: https://wpps.logisall.com
- 로그인: **P217273** (WPPS 메뉴 접근 가능 계정)
- 비밀번호: 사용자 확인 필요

## 메뉴 접근

1. 로지스올 WPPS 로그인
2. 좌측 메뉴 → **KPP(파렛트)** → 해당 메뉴 선택

## PBM110MW — 납품/반납요청 신규등록

### 주요 컬럼
| 컬럼 | 설명 | 비고 |
|------|------|------|
| col10 | 하차지 (도착지 코드) | |
| col18 | 수량 | |
| col31 | 차량번호 | |
| col32 | 기사명 | |
| col33 | 연락처 | |
| col36 | 비고 | 특이사항 |

### 처리 절차
1. 신규등록 버튼 클릭
2. 각 필드에 값 설정 (`setValue`)
3. 저장 (`fn_save`) 
4. dialog-accept 로 저장 확인
5. 검증 (`fn_search`) 으로 정상 등록 확인
6. `getValue`는 로컬캐시 참조
