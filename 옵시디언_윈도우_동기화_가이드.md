## 🎯 동기화 방법

### 방법 1: 수동 동기화 (가장 쉬움)

#### 1️⃣ 작업.bat 파일 실행
```powershell
C:\Users\%USERNAME%\옵시디언_동기화_윈도우.bat
```

#### 2️⃣ 과정
1. Git 변경사항 수집 (자동)
2. 커밋 생성 (사용자 지정 가능)
3. GitHub 푸시 (자동)
4. 로그 파일 작성

#### 3️⃣ 로그 파일
```
C:\Users\%USERNAME%\openclaw_sync_log.txt
```

---

### 방법 2: 자동 동기화 (추천)

#### 윈도우 작업 스케줄러 설정

1️⃣ **PowerShell 실행 (관리자 권한)**

```powershell
# 1. 작업 정의
$action = New-ScheduledTaskAction -Execute "C:\Users\%USERNAME%\opsync.bat"

# 2. 실행 시간 설정 (예: 매일 9AM, 6PM)
$trigger = New-ScheduledTaskTrigger -Daily -At "9:00AM"
$trigger2 = New-ScheduledTaskTrigger -Daily -At "6:00PM"

# 3. 사용자 권한 설정
$principal = New-ScheduledTaskPrincipal -UserId $env:USERNAME -LogonType Interactive -RunLevel Highest

# 4. 작업 등록
Register-ScheduledTask -TaskName "OpenClawSync" `
    -Action $action `
    -Trigger $trigger, $trigger2 `
    -Principal $principal
```

2️⃣ **스크립트 파일 생성**: `opsync.bat`
```batch
@echo off
cd /d "C:\Users\%USERNAME%\.openclaw\workspace"
git add -A
git commit -m "자동: %date% %time%"
git push origin main
```

## 📝 사용 방법 가이드

### Windows 1. **git 설치 확인**
```powershell
# 실행해보기
git --version
```
❌ 설치 안 되어있으면: https://git-scm.com/download/win

### Windows 2. **초기 저장소 설정** (한 번만)
```powershell
cd C:\Users\%USERNAME%\.openclaw\workspace
git init
git add .
git commit -m "초기 OpenClaw workspace"
```

### Windows 3. **GitHub 저장소 생성** (권장)
1. GitHub 로그인 → New repository
2. 저장소 이름: `openclaw-workspace`
3. Clone URL: `https://github.com/[사용자명]/openclaw-workspace.git`

### Windows 4. **연결 및 푸시**
```powershell
cd C:\Users\%USERNAME%\.openclaw\workspace
git remote add origin https://github.com/[사용자명]/openclaw-workspace.git
git branch -M main
git push -u origin main
```

### Windows 5. **동기화 사용**
1. Windows 탐색기에서 `옵시디언_동기화_윈도우.bat` 더블클릭
2. 커밋 메시지 입력 (옵션, 엔터만 누르면 자동 생성)
3. ✅ 완료!

## 🔍 문제 해결

### 🔴 Git 권한 문제
```powershell
# 관리자 권한 PowerShell 실행
icacls "C:\Users\%USERNAME%\.openclaw" /grant "%USERNAME%:(OI)(CI)F"
```

### 🔴 Git 커밋 실패
```powershell
cd C:\Users\%USERNAME%\.openclaw\workspace
git status                 # 현재 상태 확인
git diff                   # 변경사항 확인
git add .                  # 모든 파일 추가
git commit -m "수동 커밋"   # 커밋 재시도
```

### 🔴 Git 푸시 실패
```powershell
# SSH 키 사용 (권장)
ssh-keygen -t ed25519 -C "your_email@example.com"
# GitHub Settings → SSH and GPG keys → 새 키 추가

# GitHub 토큰 사용 (더 쉬움)
# GitHub Settings → Developer settings → Personal access tokens
git remote set-url origin https://TOKEN@github.com/[사용자명]/repo.git
```


## 🎨 자동화 팁

### 🌟 Windows 시작 시 자동 실행
```powershell
# 작업 스케줄러
$action = New-ScheduledTaskAction -Execute "C:\Users\%USERNAME%\opsync.bat"
$trigger = New-ScheduledTaskTrigger -Logon

Register-ScheduledTask -TaskName "OpenClawStartSync" `
    -Action $action `
    -Trigger $trigger
```
