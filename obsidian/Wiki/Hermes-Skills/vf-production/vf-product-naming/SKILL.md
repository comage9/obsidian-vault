# VF 제품명 naming convention (제품명 변환 규칙)

## 핵심 구분: 우드 vs 오크

| 구분 | 소재 | 색상 |
|------|------|------|
| **우드** | MDF (얇은 HDF) | 갈색 |
| **오크/내추럴오크** | 천연오크 (실내) | 자연 오크색 |
| **와이드/에센셜 우드** | MDF | **오크 색상** (밝은 오크) |

> ⚠️ 모든 우드, 오크 모두 MDF 소재

---

## 1. 맥스 系列

### 맥스 (단독)
```
보노하우스 맥스 서랍장 [색상] [단수] 1개
```
- 예: 보노하우스 맥스 서랍장 투명 5단 1개 → **540번**
- 예: 보노하우스 맥스 서랍장 다크투명 4단 1개 → **532번**

### 맥스 이동식/바퀴
```
보노하우스 국내제조 맥스 이동식 서랍장 [색상] 1개 [단수]단(규격)
```
- 예: 보노하우스 국내제조 맥스 이동식 서랍장 투명 1개 3단(360x430x705mm) → **517번**

### 어반 오크 상판 (맥스 + 오크)
```
보노하우스 어반 우드상판 베이직 서랍장 [단수]단 [색상] 1개
```
- 예: 보노하우스 어반 우드상판 베이직 서랍장 5단 투명 1개 → **542번**

### 맥스 색상 코드
| 코드 | 색상명 |
|------|------|
| O | 투명 |
| DO | 다크투명 |
| WHITE | 화이트 |
| OAK | 투명 오크 |

---

## 2. 모던플러스 系列

### 모던 (우드 = 갈색 MDF)
```
뉴모던플러스 우드형상판 [단수] [앞판색상]
```
- 예: 뉴모던플러스 우드형상판 5단 화이트 → **2번**
- 예: 뉴모던플러스 우드형상판 4단 화이트 → **666번**
- 예: 뉴모던플러스 우드형상판 5단 바이올렛 → **613번**

### 모던 (오크 = 천연오크)
```
모던플러스 내츄럴오크 우드상판형 [단수] 서랍장
```
- 예: 모던플러스 내츄럴오크 우드상판형 5단 → **1번**
- 예: 모던플러스 내츄럴오크 우드상판형 4단 서랍장 → **7번**

### 라탄 플러스 (모던 플러스 변형)
```
보노하우스 라탄플러스 우드상판 서랍장 [앞판색상] [단수]단 1개 562mm
```
- 예: 보노하우스 라탄플러스 우드상판 서랍장 그레이 6단 1개 562mm → **673번**
- 줄임: "모던 플러스 라탄 6단 그레이"

---

## 3. 에센셜/세라비/에버 라탄 系列

### 세라비 (일반/와이드)
```
보노하우스 세라비 우드상판 [와이드] 서랍장 [앞판색상] [단수]단 1개
```
- 일반: 보노하우스 세라비 우드상판 서랍장 화이트 3단 1개 → **903번**
- 와이드: 보노하우스 세라비 우드상판 와이드 서랍장 화이트 5단 1개 → **758번**

### 에버 라탄 (에센셜 라탄)
```
보노하우스 에버 라탄 우드상판 서랍장 [앞판색상] [단수]단 1개
```
- 예: 보노하우스 에버 라탄 우드상판 서랍장 화이트 3단 1개 → **775번**
- 예: 보노하우스 에버 라탄 우드상판 서랍장 버터 6단 1개 → **789번**

### 심플리 베이직
```
보노하우스 심플리 베이직 서랍장 [앞판색상] [단수]단 1개
```
- 예: 보노하우스 심플리 베이직 서랍장 3단 화이트 1개 → **932번**

### 심플리 와이드
```
보노하우스 심플리 와이드 서랍장 [단수]단 [앞판색상] 1개
```
- 예: 보노하우스 심플리 와이드 서랍장 5단 화이트 1개 → **752번**

---

## 4. 가든 라탄
```
보노하우스 가든 라탄 서랍장 [앞판색상] [단수]단 1개
```
- 예: 보노하우스 가든 라탄 서랍장 화이트 3단 1개 → **881번**

---

## 5. 로코스
```
[보노하우스] 로코스 수납 바스켓 [사이즈] [개수] [색상]
```
- 예: 보노하우스 로코스 수납 바스켓 L 라이트 화이트 4개 → **150번**
- 사이즈: L(라지), M(미디엄), S(스몰), XS

---

## 6. 색상 코드 매핑

| 코드 | 색상명 |
|------|------|
| WHITE | 화이트 |
| NAVY | 네이비 |
| NAVY2 | 네이비2 |
| GRAY | 그레이 |
| Gray2 / Light Gray | 라이트그레이 (= 그레이2) |
| Butter | 버터 |
| CREAM | 크림 |
| PINK / PINK2 / PINK3 | 핑크 |
| O | 투명 |
| DO | 다크투명 |
| BROWN | 브라운 |

## 7. 제품명 약칭 규칙

| 원본 | 약칭 |
|------|------|
| 보노하우스 맥스 서랍장 투명 6단 1개 | 맥스 6단 투명 |
| 보노하우스 국내제조 맥스 이동식 서랍장 투명 1개 3단 | 맥스 3단 투명 바퀴 |
| 보노하우스 어반 우드상판 베이직 서랍장 5단 투명 1개 | 맥스 5단 투명 오크 |
| 모던플러스 내츄럴오크 우드상판형 5단 | 모던 5단 화이트 오크 |
| 뉴모던플러스 우드형상판 4단 화이트 | 모던 4단 화이트 우드 |
| 보노하우스 세라비 우드상판 와이드 서랍장 화이트 5단 1개 | 세라비 5단 화이트 오크 (와이드) |
| 보노하우스 에버 라탄 우드상판 서랍장 버터 6단 1개 | 에센셜 라탄 6단 버터 오크 |

---

## 8. 분류 계층 구조

```
모던 플러스
├── 모던 (우드형상판 = 갈색 MDF)
│   ├── 화이트, 바이올렛, 레인보우, etc.
├── 템바보드
├── 라탄 플러스
└── 티크

에센셜
├── 심플리 (베이직)
├── 세라비
└── 에버 라탄

와이드 서랍장
├── 라탄 (가든)
├── 세라비
└── 에버 라탄

로코스
├── 바스켓 (L/M/S/XS)
├── 팬트리 리빙박스
└── 레인포그
```
