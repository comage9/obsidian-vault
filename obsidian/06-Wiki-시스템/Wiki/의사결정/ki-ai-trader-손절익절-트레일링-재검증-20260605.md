## 2. 손실 누적 원인 — "트레일링 OFF 시나리오" 문제

### 백테스트 결과 (81종목, 30일 일봉)

**트레일링 OFF 시나리오**:

| 시나리오 | 승률 | 평균 PnL |
|----------|------|----------|
| A: 익절3%/손절5% (현행) | 63.7% | **+0.16%** |
| B: 익절3%/손절3% | 56.2% | +0.37% |
| C: 익절5%/손절3% | 41.2% | +0.30% |
| D: 익절5%/손절5% | 48.8% | **-0.07%** |
| E: 익절7%/손절3% | 30.0% | -0.00% |
| F: 익절10%/손절5% | 32.5% | **-0.07%** |

**트레일링 ON 시나리오**:

| 시나리오 | 승률 | 평균 PnL |
|----------|------|----------|
| G: 익절3%/손절5%/트레일링2% | **71.2%** | +0.77% |
| **H: 익절5%/손절5%/트레일링2%** | **71.2%** | **+1.28%** |
| **I: 익절5%/손절3%/트레일링2%** | 65.0% | **+1.29%** (1위) |
| J: 익절7%/손절5%/트레일링3% | 56.2% | +0.81% |

### 결정적 인사이트

- **트레일링 OFF 시 어떤 손절/익절 조합도 평균 PnL ±0.2% 이내** (백테스트 30일, 81종목)
- **트레일링 ON 시 5~8배 개선** (G/H/I)
- **현재 운영 = A (트레일링 OFF)** → **+0.16%** (사실상 제로섬)

### "수익은 안 나고 매매만 한다"의 진짜 원인

```
매수 → 손절/익절 (트레일링 OFF)
  → 손절 -5% 또는 익절 +3% (1~3일 내 발생)
  → 손익 상쇄
  → 새로운 매수 → 반복

트레일링이 OFF면:
  +3% 수익 도달 시 즉시 익절 (작은 수익 확정)
  -5% 손실 도달 시 즉시 손절 (큰 손실 확정)
  → 손익이 상쇄되어 "매매만 함"
```
