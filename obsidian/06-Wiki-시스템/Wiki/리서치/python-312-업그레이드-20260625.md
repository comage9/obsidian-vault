## 2. 설치 방법

Ubuntu 26.04의 시스템 패키지는 Python 3.14만 제공. **uv를 사용한 격리 설치**:

```bash
# 1. .venv 백업
mv .venv .venv-3.11.bak

# 2. uv로 Python 3.12 venv 생성
uv venv --python 3.12 .venv-3.12
# → CPython 3.12.13 다운로드 + 격리 (32.5MiB)

# 3. 패키지 설치
uv pip install --python .venv-3.12/bin/python3 \
    python-dotenv pydantic sqlalchemy python-dateutil \
    beautifulsoup4 lxml psycopg2-binary requests aiohttp \
    pydantic-settings redis croniter numpy pandas neo4j \
    pixelrag
```
