### 이전: SendKeys (불안정)
- `AppActivate('Chrome')` → `SendKeys('{ENTER}')`
- OS 포커스 경쟁, COM 스레드 오류, 다국어 환경 불일치

### 현재: CDP WebUI 직접 클릭 (안정적)
```python
# chrome://print 페이지 내부 shadow DOM 경로로 [인쇄] 버튼 직접 click
root = document.querySelector('print-preview-app').shadowRoot
sidebar = root.querySelector('print-preview-sidebar').shadowRoot
button = sidebar.querySelector('print-preview-button-strip').shadowRoot
                    .querySelector('cr-button.action-button')
button.click()
```
