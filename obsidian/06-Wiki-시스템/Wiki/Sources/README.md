# Sources — 원본 자료 보관소

> **규칙:** 여기는 **수정하지 않는 원본**을 보관합니다.
> AI가 Wiki 문서를 만들 때 참고한 raw 데이터를 추적하기 위한 아카이브입니다.

## 구조

```
Sources/
├── 텍스트/         ← API raw 응답, 회의록, 로그 (Git 추적)
│   ├── LS/         ← LS API raw 데이터
│   ├── KPP/        ← WPPS raw 데이터
│   └── 기타/
├── 바이너리/       ← PDF, PNG, 스크린샷 (Git 제외, .gitignore)
│   └── README.md   ← 바이너리 파일 목록만 기록
└── README.md       ← 본 파일
```

## 규칙

| 항목 | 내용 |
|:----|:------|
| **텍스트 원본** | `.json`, `.md`, `.log`, `.csv` — Git에 포함 OK |
| **바이너리 원본** | `.pdf`, `.png`, `.jpg`, `.xlsx` — **Git에 포함 금지** |
| **파일명 형식** | `YYYY-MM-DD_설명.ext` |
| **출처 표시** | 각 파일 상단에 출처 URL 또는 시스템명 명시 |

## 용량 관리

- 텍스트만 유지 시 용량 문제 없음 (Git 3.85MiB, 여유 100MiB 목표)
- 바이너리는 `.gitignore`로 차단, 필요 시 외부 링크로 대체
- 용량 100MB 초과 시 알림 cron 작동
