# Output — AI 생성 결과물

> AI가 Wiki/Sources를 바탕으로 생성한 결과물을 날짜별로 보관합니다.
> 재사용 가능한 업무 산출물 아카이브.

## 구조

```
Output/
├── YYYY-MM-DD/     ← 날짜별 폴더
│   ├── LS-배차텍스트.md     (쿠팡 LS 배차 요청)
│   ├── KPP-등록결과.md      (KPP 팔레트 등록 내역)
│   ├── 보고서-제목.md       (위키 기반 보고서)
│   └── ...
├── 템플릿/                  ← 자주 쓰는 출력 포맷
│   ├── ls-배차요청-템플릿.md
│   └── kpp-등록-템플릿.md
└── README.md                ← 본 파일
```

## 포함 대상

| 결과물 유형 | 예시 | Git 포함 |
|:-----------|:-----|:--------:|
| 텍스트 메시지 | LS 배차 요청문 | ✅ |
| 등록 결과 보고 | KPP PBM140MW 저장 결과 | ✅ |
| 분석 리포트 | Wiki 기반 경쟁사 분석 | ✅ |
| PDF/이미지 | ❌ Output 폴더에도 저장 금지 | ❌ |

## 용량 관리

- 모든 결과물은 마크다운 텍스트 (KB 단위)
- 1,000개 결과물 기준 약 10MB 예상 — 문제 없음
- 1년 이상 지난 결과물은 `Output/Archive/`로 이동

## 사용 방법

```bash
# 새 결과물 저장 예시
cat > Output/2026-06-08/LS-배차텍스트.md << 'EOF'
# 2026-06-08 LS 배차 요청
- 1호차: 경기89바1454 / 5T / 22:00
- 2호차: 전남80바2805 / 11T / 23:50
EOF
```
