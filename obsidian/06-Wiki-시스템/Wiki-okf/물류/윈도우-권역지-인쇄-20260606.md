# 윈도우 권역지 다중 무음 인쇄

**날짜:** 2026-06-06
**분류:** 물류 > 권역지 인쇄
**대상:** Windows 10/11 (리눅스 미지원 — os.startfile 윈도우 전용)

## 개요

`E:\자주쓰는 문서\권역지\` 폴더의 권역지 PDF를 Canon G2010 series로 지정 매수 무음 인쇄한다.

## 권역지 파일 목록

| 파일명 | 권역 | 크기 |
|--------|------|------|
| DGU.pdf | 대구 | 31KB |
| EAST.pdf | 이스트 | 29KB |
| GMH.pdf | 광명 | 33KB |
| GWJ.pdf | 광주 | 31KB |
| Middle.pdf | 미들 | 32KB |
| WEST.pdf | 웨스트 | 35KB |

## 동작 방식

### 핵심 원리

`os.startfile(pdf_path, "print")` — Windows 내장 API로 PDF를 기본 프린터 스풀러에 직접 전송.
CDP, 브라우저, Adobe Acrobat 불필요. 완전 무음.

### 다중 매수 처리

PDF 라이브러리(PyMuPDF/PyPDF2) 불필요.
`os.startfile`을 N회 반복 호출 + 0.5초 간격 → 스풀러에 N개 작업 누적.

```python
import os, time, win32print

권역지_폴더 = r"E:\자주쓰는 문서\권역지"
매수 = 5  # 사용자 지정
권역명 = "east"  # dgu/east/gmh/gwj/middle/west

# 1. 프린터 확인
printer = win32print.GetDefaultPrinter()
if "Canon G2010" not in printer:
    printers = [p[2] for p in win32print.EnumPrinters(2)]
    canon = [p for p in printers if "Canon G2010" in p]
    if canon:
        win32print.SetDefaultPrinter(canon[0])

# 2. 파일명 매핑
file_map = {
    "dgu": "DGU.pdf",
    "east": "EAST.pdf",
    "gmh": "GMH.pdf",
    "gwj": "GWJ.pdf",
    "middle": "Middle.pdf",
    "west": "WEST.pdf"
}
pdf_path = os.path.join(권역지_폴더, file_map[권역명.lower()])

# 3. N회 인쇄
for i in range(매수):
    os.startfile(pdf_path, "print")
    time.sleep(0.5)
```

## 함정 (Pitfalls)

1. **매수 제한 없음** — 필요 매수만큼 반복. 10매 이상 시 `sleep(1.0)` 권장
2. **파일명 대소문자** — 모든 권역지 파일명은 대문자.pdf (EAST.pdf, DGU.pdf, ...)
3. **프린터 교정 필수** — 기본 프린터가 Canon G2010 series가 아니면 자동 교정
4. **os.startfile은 Windows 전용** — Linux에서는 사용 불가 (리눅스는 lp/lpr 사용)
