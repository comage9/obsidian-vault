# Update Template — 새 Source → 기존 Wiki 업데이트

> **용도:** 기존 Wiki 문서가 있는 상태에서 새 자료가 추가되었을 때, 기존 문서를 업데이트

## 트리거 문구
```
/update
/update [파일명]
새 자료로 기존 문서 업데이트해줘
```

## 실행 절차

### Step 1: Sources 확인 + 기존 Wiki 스캔
```bash
ls Wiki/Sources/텍스트/  # 새 파일 확인
grep -r "관련주제" Wiki/  # 기존 관련 문서 검색
```

### Step 2: AI에게 전달할 프롬프트

```markdown
새로운 Source 자료가 추가되었습니다. 다음 작업을 수행해 주세요:

1. Sources/텍스트/ 의 새 파일을 읽으세요
2. 기존 Wiki 문서 중 이 자료와 관련된 문서를 찾으세요 (index.md 참조)
3. 다음 기준으로 업데이트하세요:
   - 새로운 정보 → 기존 문서에 추가 (날짜/출처 명시)
   - 기존 정보와 충돌 → 양쪽 모두 기록하고 판단 보류
   - 새로운 개념 → 새 문서 생성
4. 업데이트한 문서와 새 문서를 [wikilink](wikilink.md)로 연결하세요
5. 각 변경사항에 출처(Source 파일명)를 반드시 표시하세요
```

### Step 3: 변경 내역 기록
- `log.md`에 `[YYYY-MM-DD HH:MM] update | [문서명] — [변경 요약]` 추가

## 예시

**사용자:** `/update`

**내 수행:**
1. `Sources/텍스트/KPP/`에 새 PBM140MW 규칙 파일 발견
2. 기존 `Wiki/물류/KPP/PBM140MW-자동화-최종-20260606.md` 검색
3. 새 규칙을 기존 문서에 추가 (변경 이력 섹션에 기록)
4. 출처: `Sources/텍스트/KPP/pbm140mw-규칙-추가-20260608.json`
5. `log.md`에 기록
