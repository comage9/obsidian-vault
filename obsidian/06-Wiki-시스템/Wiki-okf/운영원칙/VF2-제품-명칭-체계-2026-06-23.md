# VF2 제품 명칭 체계 (품목 매핑 규칙)

저장일: 2026-06-23
검증 출처: 쿠팡 발주서.xlsx (vf품목 시트), 사용자 구두 확인
신뢰도: 높음

## 목적
사용자가 약칭(예: "야채 4단 화이트", "모던 3단 화이트")으로 제품을 지칭할 때, 정확한 제품번호와 정식 명칭으로 매칭하기 위한 규칙.

---

## 1. 모던 플러스 계열 (서랍장)

### 기본 규칙
- **"모던 N단"** 이라고만 하면 → 제품번호 663 (기본형, 플라스틱 상판)
- **우드/오크/티크**를 언급하면 → 나무 상판 모델

### 제품 목록 (3단 기준)

| 제품번호 | 약칭 | 정식 명칭 | 상판 종류 | 상판 색상 |
|:--------:|:-----|:---------|:---------|:---------|
| **663** | **모던 3단 화이트** | 뉴모던플러스 3단 화이트 | 플라스틱(기본) | — |
| 19 | 모던 3단 우드 화이트 | 뉴모던플러스 우드형상판 3단 화이트 | 우드 | 흰색 |
| 12 | 모던 3단 오크 | 모던플러스 내츄럴오크 우드상판형 3단 | 오크 | 갈색 |
| 26 | 티크 3단 | 모던플러스 티크 우드상판형 3단 | 티크 | 검정색 |
| 13 | 라탄 3단 오크 | 라탄플러스 우드상판 서랍장 모던 3단 | 오크(사각형) | 갈색 |
| 932 | 에센셜 3단 화이트 | 모던 에센셜 서랍장 3단 화이트 | — | — |
| 2053 | 펭귄 바구니 | 모던 트롤리 3단 화이트 | — | — |

### 상판 색상 매핑

| 상판명 | 색상 | 비고 |
|:-------|:-----|:-----|
| **우드 (Wood)** | 흰색 | 기본 나무 상판 |
| **오크 (Oak)** | 갈색 | — |
| **티크 (Teak)** | 검정색 | — |
| (언급 없음) | — | 플라스틱 상판 (663번) |

### 라탄 플러스 특징
- 오크(갈색)만 사용
- 나무 상판이 **사각형** 형태 (다른 모델은 일반 형태)

## 4. 사용자 약칭 매칭 규칙

### 매칭 알고리즘
```
입력: "{분류} {단수}단 {색상/상판}"
→ 분류(대분류) + 단수 + 색상/상판으로 제품번호 매칭
```

### 예시

| 사용자 약칭 | 매칭 과정 | 제품번호 |
|:-----------|:---------|:--------:|
| "야채 4단 화이트" | 야채바구니 + 4단 + 화이트 | 2044 |
| "야채 3단 화이트" | 야채바구니 + 3단 + 화이트 | 2043 |
| "모던 3단 화이트" | 모던플러스 + 3단 + 화이트(기본) | 663 |
| "모던 3단 우드" | 모던플러스 + 3단 + 우드(흰색) | 19 |
| "모던 3단 오크" | 모던플러스 + 3단 + 오크(갈색) | 12 |
| "티크 3단" | 모던플러스 + 3단 + 티크(검정) | 26 |
| "라탄 3단 오크" | 라탄플러스 + 3단 + 오크 | 13 |
| "에센셜 3단" | 에센셜 + 3단 + 화이트 | 932 |
| "펭귄 바구니" | 기저귀정리함 + 트롤리 | 2053 |
| "스텝 4단 화이트" | 오픈스텝 + 4단 + 화이트 | 2040 |
