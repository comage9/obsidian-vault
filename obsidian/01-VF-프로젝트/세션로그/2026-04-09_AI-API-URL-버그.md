## VF 프로젝트 - AI API 버그 수정

### 문제
- OpenRouter API 403/404 오류
- URL: `https://openrouter.ai/api/v1/v1/chat/completions` (중복)

### 해결
- `.env`: `ANTHROPIC_BASE_URL=https://openrouter.ai` (뒤에 `/api/v1` 제거)
- `views.py`: `/v1/chat/completions` → `/api/v1/chat/completions`
- 인증: `x-api-key` → `Authorization: Bearer`
- payload: Anthropic → OpenAI format

### 테스트 결과
- `/api/ai/chat` ✅
- `/api/ai/analyze` ✅
- `/api/ai/production-recommend` ✅
- 속도: nemotron 0.45s (가장 빠른 무료 모델)
