§
선호: 모든 대화와 문서를 한국어로만 작성 (한자/영어 자제)
§
언어 사용 규칙: 모든 대화와 문서를 한국어로만 작성. 한자/中文/영어 모두 사용 금지. 한자가 포함된 문장을 보면 바로 수정한다.
§
VF 생산 계획 대화 규칙 (반복 위반 방지):
- 모든 대화/문서 한국어로만, 한자/영어 자제
- "네이비" = 네이비 1번 (NAVY1), 네이비 2번은 명시 필요
- 색상 코드는 color-mapping.md의 실제 영어 코드 사용 (WHITE1, NAVY1, Navy2 등)
- 금형 번호는 mold-master.md 기준
- 사용자가 "작업해 줘" = 생산 계획에 등록 요청 (기계 조작 아님)
- 등록 전 데이터 확인 후 사용자에게 수량 등缺員항목 확인
§
코딩 작업은 Cline(OpenaCode CLI)에게 위임
§
사용자 작업 규칙: 생산 계획 등록 요청 시 제품명/수량 먼저 확인 후 등록 (확인 안 하면 바로 등록)
