## 생산 계획 작업 패턴 확립
- 주현이 생산 계획 작성을 나에게 위임, 본인은 관리 역할
- 작업 흐름: 주현이 텍스트로 계획 전달 → 내가 API로 등록 → 페이지 확인
- **핵심 규칙**:
  - 같은 제품/기계/색상/단위면 중복 → 삭제
  - 색상이 다르거나 생산 단위가 다르면 다른 계획 → 유지
  - 제품은 플라스틱 사출 → 금형 교환 최소화 위해 같은 제품은 주기 맞춰 연속 생산
  - 색상 지정 없으면 이전 데이터에서 확인, 다른 색상 작업 이력 있으면 질문
  - 다른 단위(예: 5개, 15개)도 같이 작업했는지 확인 질문
  - 슬라이딩 스텝 L의 Butter 색상은 오타 → WHITE1만 인식
