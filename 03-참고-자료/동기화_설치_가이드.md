## 🔄 자동 동기화 설정 (선택)

### 1. 작업 스케줄러 등록

#### PowerShell 스크립트:
```powershell
# 스크립트 파일 생성
$scriptContent = @"
@echo off
cd C:\Users\comage\.openclaw\workspace
.\optsync.bat pull
"@
Set-Content -Path "C:\Users\comage\sync_morning.bat" -Value $scriptContent

# 작업 스케줄러 등록
$action = New-ScheduledTaskAction -Execute "C:\Users\comage\sync_morning.bat"
$trigger = New-ScheduledTaskTrigger -Daily -At 9am
$principal = New-ScheduledTaskPrincipal -UserId "comage" -LogonType Interactive -RunLevel Highest
Register-ScheduledTask -TaskName "OpenClawSync_Morning" -Action $action -Trigger $trigger -Principal $principal
```

#### 실행 방법:
```powershell
# PowerShell을 Administrator 로 실행하고 위 스크립트 붙여넣기
```

### 2. 수동 자동화 (PowerToys)
Windows PowerShell 에서:
```powershell
# 빠른 실행 단축키 설정
New-Item -ItemType SymbolicLink -Path "C:\Users\comage\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\StartUp\Sync.lnk" -Target "C:\Users\comage\.openclaw\workspace\optsync.bat"
```

## 🔧 문제 해결

### 1. "Git 이 아닙니다" 오류
```cmd
# Git 설치 확인
git --version

# Git PATH 추가
cd C:\Program Files\Git\bin
```

### 2. 인증 오류
```cmd
# Git Credential Manager 확인
git credential manager

# 새 토큰 생성: https://github.com/settings/tokens
# optsync.bat 의 GITHUB_TOKEN 상수 업데이트
```

### 3. 충돌 발생
```cmd
# 충돌 해결 후
git add .
git commit -m "충돌 해결"
git push
```
