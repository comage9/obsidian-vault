# Windows 동기화 사용 가이드 (실제 실행 방법)

## ❗ 문제: 'optsync.bat' 용어가 인식되지 않음

### 원인
현재 PowerShell 디렉토리가 `C:\Users\kis\obsidian-vault` 일 때,
`optsync.bat` 파일은 `C:\Users\kis\.openclaw\workspace` 에 있습니다.

### ✅ 해결 방법 1: 디렉토리 이동 (권장)

```powershell
# 1. 올바른 디렉토리로 이동
cd C:\Users\kis\.openclaw\workspace

# 2. 현재 위치 확인
pwd

# 3. optsync.bat 실행
.\optsync.bat

# 다른 옵션
.\optsync.bat pull    # Pull 만
.\optsync.bat push    # Push 만
.\optsync.bat status  # Git 상태 확인
.\optsync.bat log     # 최근 커밋 확인
.\optsync.bat diff    # 변경 파일 확인
.\optsync.bat all     # 전체 작업
```

### ✅ 해결 방법 2: 경로 직접 지정

```powershell
# 경로 전체로 실행
C:\Users\kis\.openclaw\workspace\optsync.bat

# 옵션
C:\Users\kis\.openclaw\workspace\optsync.bat pull
C:\Users\kis\.openclaw\workspace\optsync.bat push
C:\Users\kis\.openclaw\workspace\optsync.bat status
C:\Users\kis\.openclaw\workspace\optsync.bat log
C:\Users\kis\.openclaw\workspace\optsync.bat diff
C:\Users\kis\.openclaw\workspace\optsync.bat all
```

### ✅ 해결 방법 3: PowerShell 에 Alias 추가 (추천)

```powershell
# 이번 세션에서만 사용 가능
Set-Alias -Name optsync -Value "C:\Users\kis\.openclaw\workspace\optsync.bat"

# 이제 어디서든 실행 가능
optsync.bat pull
optsync.bat push
optsync.bat status
optsync.bat log
optsync.bat diff
optsync.bat all

# 영구적으로 사용하려면 profile 에 추가
echo "Set-Alias -Name optsync -Value 'C:\Users\kis\.openclaw\workspace\optsync.bat'" >> $PROFILE

# PowerShell 재시작 후 항상 사용 가능
```

## 🚀 자동화 설정

### PowerShell Alias (권장)
```powershell
# Profile 에 추가
echo '
# OptSync Alias
Set-Alias -Name optsync -Value "C:\Users\kis\.openclaw\workspace\optsync.bat"
' >> $PROFILE

# 적용
Get-Content $PROFILE | Invoke-Expression

# 테스트
optsync.bat status
```

### 작업 스케줄러 (선택)
```powershell
# 자동 동기화 스크립트 생성
$scriptContent = @"
@echo off
cd C:\Users\kis\.openclaw\workspace
.\optsync.bat pull
"@
Set-Content -Path "C:\Users\kis\sync_morning.bat" -Value $scriptContent

# 작업 스케줄러 등록
$action = New-ScheduledTaskAction -Execute "C:\Users\kis\sync_morning.bat"
$trigger = New-ScheduledTaskTrigger -Daily -At 9am
$principal = New-ScheduledTaskPrincipal -UserId "kis" -LogonType Interactive -RunLevel Highest
Register-ScheduledTask -TaskName "OpenClawSync_Morning" -Action $action -Trigger $trigger -Principal $principal
```
