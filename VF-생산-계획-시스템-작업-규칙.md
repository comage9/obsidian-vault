## 🎯 핵심 규칙 요약

### 1. 두 가지 유형의 생산 제품

#### A. **금형으로 기계에서 만드는 제품** (기계 지정 가능)
- **특징**: 특정 기계에서 금형으로 직접 생산
- **예시**: 
  - "모던 플러스 서랍"
  - "모던 플러스 프레임" 
  - "모던 플러스 앞판"
  - "블랑 투명"
- **규칙**: 
  - 일반적으로 기계 번호가 고정되어 있음
  - **BUT**: 반드시 기계 번호가 필수는 아님
  - 생산 대기 상태로 기계 번호를 지정할 수도 있음
  - 상황에 따라 유연하게 처리 가능

#### B. **제품번호가 있는 완제품** (기계 지정 불필요)
- **특징**: 여러 기계에서 나온 부품이 조립/가공된 완제품
- **예시**:
  - "모던 플러스 4번" (제품번호 4)
  - "모던 플러스 5번" (제품번호 5)
  - "모던 플러스 13번" (제품번호 13)
  - "모던 플러스 6번" (제품번호 6)
  - "로코스 M 화이트 네 개"
- **규칙**: 기계 번호를 지정하지 않음

## 🔧 상세 작업 규칙

### 1. 기계 번호 지정 규칙

#### ✅ **기계 번호 지정하는 경우**
- 제품명에 **금형 정보가 포함된 경우** (일반적)
- 예: "모던 플러스 서랍", "모던 플러스 프레임", "모던 플러스 앞판"
- **BUT**: 반드시 필수는 아님, 상황에 따라 다름

#### ❌ **기계 번호 지정하지 않는 경우**
- 제품명에 **숫자 번호가 포함된 경우** (4번, 5번, 13번, 6번 등)
- 예: "로코스 M 화이트 네 개 작업해 줘"
- **특별히 지시하지 않으면 기계 번호 지정하지 않음**
- 이유: 기계 위치가 바뀔 수 있음

#### ⚠️ **유연한 처리**
- 금형 제품도 생산 대기 상태로 기계 번호를 지정할 수 있음
- 절대적인 규칙보다 상황에 맞는 판단 필요
- 사용자 지시에 최우선 순위 부여

### 2. 데이터 입력 규칙

#### A. 금형 제품 (기계 지정 필수)
```json
{
  "machineNumber": "4",      // 고정된 기계 번호
  "moldNumber": "2",         // 금형번호
  "productName": "모던플러스 서랍"
}
```

#### B. 완제품 (기계 지정 불필요)
```json
{
  "machineNumber": "0",      // 0 또는 임의값 (중요하지 않음)
  "moldNumber": "4",         // 제품번호
  "productName": "모던 플러스 4번"
}
```

## 📋 실제 작업 예시

### 예시 1: 금형 제품 (기계 지정 가능)
**요청**: "모던 플러스 서랍 생산해 줘"
**처리**:
- 일반적: 기계번호 4번 (고정)
- 대안: 생산 대기 상태로 기계번호 지정 가능
- 금형번호: 2번
- 제품명: "모던플러스 서랍"
- **주의**: 반드시 기계번호 지정이 필수는 아님

### 예시 2: 완제품 (기계 지정 안함)
**요청**: "로코스 M 화이트 네 개 작업해 줘"
**처리**:
- 기계번호: 지정하지 않음 (0 또는 임의값)
- 금형번호: 제품번호 사용 (예: 268)
- 제품명: "로코스 M 화이트 네 개"

### 예시 3: 숫자 제품 (기계 지정 안함)
**요청**: "모던 플러스 4번 한 팔레트"
**처리**:
- 기계번호: 지정하지 않음
- 금형번호: 4 (제품번호)
- 제품명: "보노하우스 라탄플러스 우드상판 서랍장 모던 화이트 6단"

| 모던 플러스 3번 | 3 | 완제품 | ❌ 일반적으로 아니오 |

| 모던 플러스 서랍 | 2 | 금형제품 | ⚠️ 상황에 따라 (일반적: 기계 4번) |
| 모던 플러스 프레임 | 101 | 금형제품 | ⚠️ 상황에 따라 (일반적: 기계 5번) |

## 💡 중요 참고사항

1. **기계 위치 변동성**
   - 완제품은 여러 기계에서 작업될 수 있음
   - 기계 위치가 바뀔 수 있으므로 고정하지 않음
   - 특별 지시가 없으면 기계번호 지정하지 않음

2. **제품번호 vs 금형번호**
   - **제품번호**: 완제품 식별용 (4, 5, 13, 6 등)
   - **금형번호**: 특정 기계의 금형 식별용 (2, 101, 5 등)
   - 같은 숫자라도 의미가 다를 수 있음

3. **사용자 요청 해석**
   - 숫자가 포함된 제품명 → 완제품 → 일반적으로 기계번호 지정 안함
   - 부품명이 포함된 제품명 → 금형제품 → **상황에 따라** 기계번호 지정 가능
   - **BUT**: 금형 제품도 반드시 기계 번호 지정이 필수는 아님
   - 생산 대기 상태로 기계 번호 지정할 수 있음
   - 명확하지 않으면 확인 요청
