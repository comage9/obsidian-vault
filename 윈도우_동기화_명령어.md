## ⏰ 자동 동기화 설정 (작업 스케줄러)

### 1. 배치 파일 생성
아래 내용을 `C:\Users\comage\sync_openclass.bat` 에 저장:
```batch
@echo off
cd C:\Users\comage\.openclaw\workspace
git pull origin main
git add .
git commit -m "auto sync: %date% %time%"
git push origin main
```

### 2. 작업 스케줄러 등록
```powershell
# PowerShell 에서 실행
$action = New-ScheduledTaskAction -Execute "C:\Users\comage\sync_openclass.bat"
$trigger = New-ScheduledTaskTrigger -Daily -At "9:00AM"
$principal = New-ScheduledTaskPrincipal -UserId "comage" -LogonType Interactive -RunLevel Highest
Register-ScheduledTask -TaskName "OpenClawSync" -Action $action -Trigger $trigger -Principal $principal
```

## 📝 메모리 시스템

### 인덱스 구조
1. **MEMORY.md** - 주요 프로젝트 및 결정 사항
2. **memory/index.md** - 모든 메모리 파일 인덱스
3. **memory/2026-MM-DD.md** - 일일 작업 기록

### 자동 업데이트
- 모든 작업은 `memory/2026-MM-DD.md` 에 기록
- 주요 결정 사항은 `MEMORY.md` 에 업데이트
- 인덱스 파일 (`index.md`) 에 날짜 링크 추가

### Git 동기화 규칙
- **매일 작업 종료 후**: `memory/2026-MM-DD.md` Git Push
- **주간 검토**: 금요일 전체 Git Sync
- **월간 정리**: 매월 말 정리 및 보관
