### 날짜 자동 이동 로직 (매우 중요)
**요청**: 종료(ended) 아닌 생산 계획은 자동으로 다음 날로 이동
- 이전 날짜 미완료 → 오늘(또는 다음 근무일)으로
- 오늘 미완료 → 내일(또는 다음 근무일)으로
- 주말이면 다음 평일로 자동 스킵

**구현 위치**: `backend/sales_api/views.py` - `ProductionPlanViewSet.list()`
**로직**: 
```python
# 이전 날짜의 ended 아닌 항목 → 다음 근무일로
# 오늘의 ended 아닌 항목 → 내일(또는 다음 근무일)로
```

**결과**: 
- 4/10 ended 아닌 41개 → 4/13(월요일)로 자동 이동
- 4/11 ended 아닌 항목 → 4/13으로 이동
- 최신 날짜가 4/13으로 표시됨
