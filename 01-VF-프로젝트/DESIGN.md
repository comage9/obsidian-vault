# 보노하우스 VF 프로젝트 DESIGN.md

## 개요

VF(Vision Furniture) 프로젝트는 보노하우스의 모듈형 가구 라인입니다.
현재 Website VF Production(http://bonohouse.p-e.kr:5174)에서 생산 관리 중.

## 제품 분류

### 시리즈
- **모던** — 모던+내추럴오크우드상판형. 기본형.
- **맥스** — 맥스+서랍장투명/어반우드상판베이직. 고급형.

### 단수
- 1단 ~ 6단 (제품명에 '[단수]'로 표시)

### 색상 코드 (color-mapping.md 기준)

| 코드 | 의미 |
|------|------|
| WHITE1 | 화이트 |
| NAVY1 | 네이비 1번 |
| Navy2 | 네이비 2번 |
| LIGHTGRAY | 라이트그레이 |
| LIGHTGRAY2 | 그레이2 (라이트그레이와 동일) |

## 색상 규칙

네이비 = NAVY1 (네이비 1번)만 사용. 네이비 2번은 'Navy2'로 명시.

## 금형 번호

금형 번호는 mold-master.md 참조.

## 제품명 규칙

```
모던[단수][앞판색상]오크=모던플러스내추럴오크우드상판형[단수]
맥스[단수]투명=맥스서랍장투명[단수]단
맥스[단수]투명오크=어반우드상판베이직서랍장[단수]단투명
```
